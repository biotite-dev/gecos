Install instructions
====================

Requirements
------------

*Gecos* requires at least *Python* version 3.5. *Gecos* can be installed
via

.. code-block:: none

   pip install gecos

from PyPI or via

.. code-block:: none

   pip install .

from a local source directory.
Note that the following packages are required:

   - **biotite**
   - **numpy**
   - **matplotlib**
